AdminLedger for Jira

Customer Guide

What AdminLedger Does

AdminLedger scans Jira administration metadata and creates a prioritized review queue for:

  • Custom field inventory risk.
  • Workflow cleanup candidates.
  • Archived project review.
  • Admin-like group sprawl.
  • Optional Confluence report documentation freshness.
  • Directional monthly waste estimates.

First Scan

  1. Install AdminLedger from the Atlassian Marketplace.
  2. Open AdminLedger from Jira admin apps or the global app page.
  3. Select Run scan.
  4. Review the health score, findings queue, and recommended review.
  5. Export CSV or publish a Confluence report if needed.

Settings

Setting Purpose
Monthly Jira seat cost Used for directional monthly waste estimates.
Inactivity review window Used for license and access-review policy signals.
Confluence space ID Enables Confluence report publishing into the selected space.
Weekly scheduled scan Enables the scheduled scan trigger.

CSV Export And Confluence Publishing

Use Export CSV to download findings for audit review, cleanup planning, or finance handoff. To publish Confluence reports, add a Confluence space ID in settings, save settings, run a scan, then select Publish report.
